This is a bash script file which helps to download all required softwares and updates in ubuntu
How to run:
	  1. Download Script File
	  2. Open Terminal (Ctrl+Alt+T)
	  3. cd Downloads
	  3. mv install.sh ~
	  4. chmod 777 install.sh
	  5. bash install.sh
